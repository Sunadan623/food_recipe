# Food Recipe App

A simple Food Recipe Application that allows users to search, explore, and view recipes.  
Built for learning and practicing web development concepts.

---

## Features

- Search recipes by dish name or ingredients
- View detailed cooking instructions
- List of required ingredients
- Save favorite recipes
- Responsive and easy-to-use interface

---

## Tech Stack

- Frontend: React / Next.js
- Backend: Node.js / Express
- Database: MongoDB
- API: Spoonacular API (or custom recipe API)
